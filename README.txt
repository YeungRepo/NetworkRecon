### This repository contains code for the paper entitled "Discovery of Data-Driven Network Models for Genetic Circuits From Time-Series Data with Incomplete Measurements" by Enoch Yeung, Jongmin Kim, Ye Yuan, Jorge Goncalves, and Richard Murray, submitted to biorXiv, with URL:  

https://www.biorxiv.org/content/10.1101/2021.03.10.434835v2.    

More generally, it contains algorithms and data for working examples of dynamical structure reconstruction, a theory first piloted by Sean Warnick and Jorge Goncalves in the IEEE Transactions of Automatic Control paper: 

"Necessary and Sufficient Conditions for Dynamical Structure Reconstruction of LTI Networks"

URL: https://ieeexplore.ieee.org/document/4623259
